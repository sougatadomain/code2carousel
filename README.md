# 📚 AI Study Buddy

AI Study Buddy is an intelligent, full-stack learning companion designed to help students optimize their study time. Powered by Google's Gemini AI, this application transforms raw study materials into interactive quizzes, concise summaries, and structured learning paths.

**View the core app in AI Studio:** [AI Study Buddy Project](https://ai.studio/apps/1eec7c9c-ceb7-4304-8d3c-74705ebdd1fc)

---

## ✨ Features
* 📝 **Smart Note Synthesis:** Instantly condense long lecture notes into readable, actionable bullet points.
* ❓ **Dynamic Quiz Arena:** Turn your study materials into interactive practice tests to check your retention.
* ⏱️ **Focus Timer:** Built-in Pomodoro timer to ensure distraction-free study loops.
* 📊 **Analytics Dashboard:** Track your live study streaks and monitor your learning momentum.

## 🛠️ Tech Stack
* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **AI Integration:** Google Gemini API 
* **Deployment:** Vercel
