# 🎠 Code2Carousel AI

Code2Carousel AI is a full-stack web application designed for developers and content creators. It seamlessly transforms raw code snippets into beautiful, engaging, and swipeable carousels perfect for LinkedIn, Instagram, and Twitter. Powered by Google's Gemini AI, it not only formats your code but automatically generates smart explanations for each slide.

---

## ✨ Features
* 🤖 **AI-Powered Breakdowns:** Paste your code and let Gemini AI automatically split it into logical steps with clear explanations.
* 🎨 **Stunning Visuals:** Beautiful, customizable themes with syntax highlighting tailored for social media feeds.
* 📏 **Responsive Canvas:** Perfectly sized export ratios for LinkedIn (PDF) and Instagram (PNGs).
* ⚡ **Real-Time Preview:** Watch your carousel generate and adjust dynamically as you edit the code or prompts.

## 🛠️ Tech Stack
* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **AI Integration:** Google Gemini API
* **Deployment:** Vercel
